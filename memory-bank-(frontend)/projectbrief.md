# Proje Özeti (frontend)

## Amaç
Akıllı Triage System için kullanıcı odaklı, hızlı ve anlaşılır web arayüzünü geliştirmek.

## Kapsam
- Login ekranı (rol bazlı yönlendirme)
- Personel paneli (hasta bul/oluştur, triyaj, override, kayıtlar)
- Admin paneli (kayıtlar, filtreler, dataset, export)
- API kontratına tam uyum

## Topoloji
- Tek repo içinde frontend modülü
- Şu an local sistem için geliştirilecek, sonradan sunucuya taşınabilir şekilde yapılandırılacak

## Başarı Kriteri
- Personel ve Admin ana akışları UI'da uçtan uca tamamlanır
- Hata mesajları, form validasyonları ve yüklenme durumları net görünür
